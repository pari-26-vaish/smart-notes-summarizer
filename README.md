# 📝 Smart Notes Summarizer

Smart Notes Summarizer is a web-based application that automatically condenses long notes, articles, or documents into short, meaningful summaries — helping users save time and grasp key information quickly.

## 🚀 Features
- Paste or upload text/notes for summarization
- Generate concise, accurate summaries
- Support for multiple input formats (text, PDF, etc.)
- Adjustable summary length (short/medium/long)
- Clean and simple, responsive user interface

## 🛠️ Tech Stack
**Frontend:**
- HTML5
- CSS3
- JavaScript

**Backend:**
- Java
- Spring Boot
- Spring MVC / Spring Data JPA

**Database:**
- MySQL (or your DB of choice)

**Build Tool:**
- Maven
